# Random User Generator

A simple web app that generates random user details using HTML, CSS, JavaScript, and the Random User API.

## Features

- Generate random user profiles
- Shows user image
- Shows full name
- Shows email address
- Shows location
- Clean responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
- Random User API

## How to Use

1. Open the website
2. Click the **Generate User** button
3. A new random user profile will appear
